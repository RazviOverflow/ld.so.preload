Title: Instructions for compilation
	Download the source code from <here: https://github.com/RazviOverflow/ld.so.preload>.

	In order to compile the source code you must use Make. From the root project directory execute:

	- *make* if you want to compile the code in _info_ mode. Info mode will simply log error message in the logging files.
	- *make debug* if you want to compile the code in _debug_ mode. Debug mode will log all messages in the logging files, not only errors. 

	Further instructions about using the library are given at <the main project's README: https://github.com/RazviOverflow/ld.so.preload/blob/master/README.md>. This documentation is exclusively about the code.
